# Swellscan

> Every inbox is a shore. We scan every swell that hits it.

A Gmail Add-on that scores inbound email for maliciousness — using layered detection:
cheap, deterministic heuristics first, with LLM reasoning reserved for the cases
where the signal is genuinely ambiguous.
